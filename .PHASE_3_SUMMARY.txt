🌐 WEBSOCKET IMPLEMENTATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Conexión:
  ├─ Cliente: new WebSocket("ws://localhost:8000/ws")
  └─ Gateway: websocketHandler()
      ├─ Upgrade HTTP → WebSocket
      ├─ Registra cliente en pool
      └─ Envía mensaje de bienvenida

Broadcasting (Real-Time):
  ├─ Redis: publish("agency_events", eventJSON)
  ├─ Gateway: subscribeToRedisEvents()
  ├─ Callback: broadcastEvent()
  ├─ Pool: iterate all clients
  ├─ Goroutine: send to each client
  └─ Log: broadcast completado

Desconexión:
  ├─ Client cierra o timeout
  ├─ ws.ReadJSON() error
  └─ Remove from pool + cleanup
