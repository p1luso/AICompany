# Frontend - Dashboard Visual

## 📌 Descripción

Interfaz Next.js para la oficina virtual con 3 áreas principales:

### Secciones

1. **Input**: Formulario para enviar tareas a la agencia
2. **"La Oficina" (Live Feed)**: Chat en tiempo real
   - Consumidor WebSocket
   - Avatares y colores únicos por agente
   - Conversación Manager → Especialista → QA
3. **Output**: Panel con resultado final formateado

## 📦 Stack

- **Next.js**: App Router
- **React**: UI components
- **Tailwind CSS**: Styling
- **Zustand**: State management
- **WebSocket**: Real-time streaming

## 🎨 Diseño

- Dashboard tema oscuro
- Responsive layout
- Reconexión automática WebSocket
- Animaciones fluidas
